Regresin Polinomial y Modelos de Prediccin: Tiempo vs Calificacin

Este proyecto realiza un anlisis y limpieza de datos del archivo "A1.6 Tiempo de entrega.csv". Despues, genera grficos para visualizar la relacin entre el tiempo de entrega y la calificacin obtenida.

1. Carga y Exploracin de Datos

  • Se muestran las dimensiones del conjunto de datos.
  • Se eliminan espacios innecesarios en los nombres de las columnas.
  • Se imprime el tipo de datos de cada columna para detectar posibles inconsistencias.

Limpieza de Datos

La limpieza se realiza en varias etapas:

  • Se convierten las columnas Tiempo y Calificacin a formato numrico.

  • Se detectan y eliminan filas con valores no numricos.

  • Se eliminan registros donde Tiempo o Calificacin sean menores a 0.

  • Se establece un lmite superior de 110 para Calificacin.

  • Se calcula el rango intercuartlico (IQR) para Tiempo.

  • Se eliminan valores extremos utilizando un umbral amplio de k=20 para evitar eliminar datos relevantes.

Visualizacin de Datos

Se generan dos grficos de dispersin con matplotlib:

  • Antes de la limpieza: Muestra los datos crudos, incluyendo posibles valores atpicos.
  • Despus de la limpieza: Presenta los datos ya filtrados, proporcionando una vista ms clara de la tendencia.

Prediccin del Tipo de Algoritmo

Antes de realizar los modelos mi prediccin es que el modelo de segmentos sera el mejor para encontrar mejores predicciones de calificaciones.

3. Regresin Polinomial

Este cdigo entrena un modelo de regresin polinomial utilizando los datos de entrenamiento. Se elige un grado 2 (degree=2), lo que permite capturar relaciones no lineales entre el tiempo de entrega y la calificacin.

El modelo sugiere que existe una relacin no lineal entre el tiempo de entrega (x1, x2) y la calificacin.

  • Coeficiente de x1 (trmino lineal) = 0.3859 Es positivo y estadsticamente significativo (p < 0.001), lo que indica que, en promedio, un mayor tiempo de entrega est asociado con un ligero aumento en la calificacin.
  • Coeficiente de x2 (trmino cuadrtico) = -0.0018 Es negativo y significativo (p = 0.029), lo que sugiere una posible curvatura en la relacin: inicialmente, la calificacin aumenta con el tiempo, pero luego tiende a disminuir para valores ms altos.

5. Regresin KNN

Se entrena un modelo de regresin K-Nearest Neighbors (KNN) utilizando KNeighborsRegressor de sklearn. La cantidad de vecinos (k) se establece en 5, aunque este valor puede ajustarse segn el comportamiento de los datos.

Este enfoque es til cuando la relacin entre las variables es compleja o no lineal.

Conclusin

Este anlisis busca determinar si existe una relacin entre el tiempo de entrega y la calificacin obtenida. A travs de diferentes modelos de prediccin, se exploraron patrones en los datos para evaluar si entregar una tarea ms cerca de la fecha lmite afecta el desempeo acadmico.

Los resultados muestran que existe una relacin, aunque no es completamente lineal. Algunos modelos sugieren que las calificaciones pueden verse afectadas en ciertos rangos de tiempo, pero otros factores tambin influyen en los resultados.

En trminos de desempeo, la regresin segmentada fue el modelo con mejor ajuste a los datos, aunque cada enfoque tiene ventajas y limitaciones dependiendo del contexto.
